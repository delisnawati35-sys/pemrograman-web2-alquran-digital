# Sistem Pembelajaran Al-Qur'an Digital

## Deskripsi
Website Sistem Pembelajaran Al-Qur'an Digital yang dibuat untuk memenuhi tugas UAS Pemrograman Web 2.

## Fitur
- Halaman Beranda
- Materi Pembelajaran
- Data Pembelajaran Al-Qur'an
- Login Admin
- Manajemen Data

## Teknologi
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Cara Menjalankan
1. Jalankan Apache dan MySQL pada XAMPP.
2. Import database ke phpMyAdmin.
3. Simpan project pada folder htdocs.
4. Buka browser.
5. Akses website melalui localhost.
